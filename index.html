CSS:

Here we are going to show some common CSS properties for text! Open the "demostyle.css" and follow us!

The color property is used to set the color of the text. which can be specified by

  • a color name - like "red"
  • a HEX value - like "#ffffff"
  • an RGB value - like "rgb(255,255,0)"

Assign a color in #color-demo in the CSS to see how this line changes!

The text-align property is used to set the horizontal alignment of a text. A text can be left or right aligned, center, or justify.

Try different text-align values in #align-demo in the CSS to see how this line changes!

The text-decoration property is used to set or remove decorations from text. Common available values are none, overline, line-through, or underline.

Try different text-decoration values in #decoration-demo in the CSS to see how this line changes!

The text-transform property is used to specify uppercase and lowercase letters in a text. Common available values are uppercase, lowercase, or capitalize.

Try different text-transform values in #transform-demo in the CSS to see how this line changes!

The text-indent property is used to specify the indentation of the first line of a text.

Try different values in #indent-demo in the CSS to see how this line changes!

The text-shadow property is used to adds shadow to text.
It consists of 4 values: text-shadow: horizontal shadow, vertical shadow, blur effect range, color of shadow;

Try different values in #shadow-demo in the CSS to see how this line changes!

HTML/CSS-OTHERS:

As you may have noticed, we used div in the HTML file as a container for each parts. We could put more than one HTML element in the division, group together elements in div and can apply CSS for them. The div tag is very flexible in creating web layouts and easy to modify.

Brief introduction to Box Model

The CSS box model is essentially a box that wraps around every HTML element. It consists of: borders, padding, margins, and the actual content.

Attention: when we set the width and height values for an element, we are setting the width and height of the content area. We also need to add padding, borders and margins when calculating the whole size of the element.

You can define margin by margin-top, margin-right, margin-bottom, margin-left respectively where the values could be auto(horizontally center the element within its container.), px, cm, etc., % of the width of the containing element, or inherited from the parent element. Or you can use a margin shorthand: margin: top-margin, right-margin, bottom-margin, left-margin. Padding follows the similar rule.
